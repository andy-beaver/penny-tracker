Penny War Tracker

Instructions

Welcome to the Penny War Tracker! This application helps you manage and track a Penny War competition, complete with setup, daily entries, and visualizations.

  1. Setup: Use the Setup Page to configure the start and end dates, teams, colors, and initial points.
  2. Daily Entries: Use the Daily Entry Page to record daily updates for each team's points.
  3. Visualization: Use the Visualization Page to see a graphical representation of the current standings.
